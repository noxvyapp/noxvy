# 🚀 NOXVY

**Noxvy** é uma empresa de tecnologia focada em desenvolver soluções de software (SaaS) premium, impulsionadas por Inteligência Artificial, para acelerar e automatizar negócios.

---

## 💻 Nosso Ecossistema

### 1. Capturo
**Máquina de Prospecção com IA**
Sistema completo de CRM e prospecção B2B. Inclui Radar de leads, funil Kanban, e geração automatizada de pitches e landing pages utilizando inteligência artificial.
🔗 **Acesso:** [capturo.noxvy.com.br](https://capturo.noxvy.com.br)

### 2. Kronos (Anteriormente Vektor)
**Inteligência para Campanhas e Afiliados**
Sistema avançado de análise de campanhas, garimpagem de produtos, criador de estruturas e presells com integração direta à API do Google Ads.
🔗 **Acesso:** [kronos.noxvy.com.br](https://kronos.noxvy.com.br) *(Em estruturação)*
